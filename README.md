# NEXUS Intercom — sito di presentazione

Pagina pubblica di presentazione di **NEXUS Intercom**, la matrice intercom
broadcast per iPhone, iPad e Mac: TALK, LISTEN, partyline, IFB e fino a 16 N-1
differenziati dai canali di una Dante Virtual Soundcard.

Sito statico (una pagina, `index.html`) — nessun codice del prodotto, solo
presentazione. Gli screenshot sono autoscatti dell'app con dati dimostrativi.
